How does the AI prediction work?

The AI uses a Gradient Boosting Regressor to predict how many seconds the next bus will arrive earlier or later than the LTA API estimate.

Input features

  • Time of day — sin/cos cyclical encoding
  • Day of week — weekday / weekend flag
  • Peak hour — Mon–Fri 7–9 am, 5–7 pm
  • Bus load — SEA / SDA / LSD
  • Bus type — SD / DD / BD
  • Route & stop hash — numeric proxy

Ground truth

The background collector tracks each bus across API polls. When a bus disappears it has arrived. delay = last_estimate − first_estimate. This data retrains the model hourly.

Bootstrap

On first run the model uses 5 000 synthetic rows encoding known SG patterns (peak congestion, late-night reliability). Real data blends in with 3× weight as it accumulates.
